Цель:
Не дать противникам дойти до базы (отмечена зелёным тайлом)

Управление:

ПКМ - перемещение карты
Колёсико мыши - приближение/отдаление
ЛКМ - постройка активной башни на клетке
1 - выбор базовой башни как активной
2 - выбор мортиры как активной
3 - выбор снайперской бани как активной

Бызовая информация:
Наземные противники перемещаются по дороге
Воздушные противники перемещаются по любым тайлам

Все башни атакуют ближайшие к ним цели

Базовая башня - атакует все цели
На последнем уровне делает дополнительный выстрел по той же цели

Снайперская башня - атакует все цели
Имеет шанс на мнгновенное убийство

Мортира - атакует только наземные цели
Наносит урон по площади
